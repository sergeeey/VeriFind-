# VeriFind Arena

Financial debate system with multi-agent AI analysis.

## Features
- Real-time debate visualization
- Bull vs Bear perspectives
- Market data integration
- Compliance tracking

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

```bash
npm install
npm run dev
```

## Architecture
Based on APE 2026 debate system.
